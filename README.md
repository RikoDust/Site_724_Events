# Site_724_Events


## 📋 À propos

Projet de formation consistant à repérer et fixer l'ensemble des bugs du site d'une agence évènementielle. Rédiger un cahier de recettes suite au débogage.



Contexte : Projet de formation (Openclassrooms)

## ✨ Fonctionnalités principales

- Un page avec affichage dynamique de contenue
- Carrousel d'evennements
- Présentation de l'agence 724 Events
